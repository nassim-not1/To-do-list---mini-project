#  To-Do List & Présentation React

Ce dépôt contient le code source de l'application ainsi que le support de la présentation sur React JS.

##  Organisation du Projet

* **`to-do-list/`** : Le dossier principal de l'application.
  * **`src/`** : Contient le code source (composants, logique et styles).
* **`React_JS-Presentation.pptx`** : La présentation des concepts clés de React.

##  Fichiers de Configuration

À la racine du projet, vous trouverez plusieurs fichiers techniques (tels que `package.json`, `vite.config.js`, `index.html` ou `.gitignore`).

**Pourquoi ces fichiers sont importants :** Ils constituent le moteur du projet. Ils servent à gérer les outils de développement, les bibliothèques nécessaires et la manière dont l'application est compilée. Ces fichiers sont indispensables au fonctionnement du projet et ne doivent pas être supprimés.
